Detailed Description

Static tests methods for the BigInt class.

These methods provide some quick and simple basics tests. Each one of them instanciates some BigInt objects and cout some infos so you can see if results are wrong.

Those tests are far from perfect, it's just basic stupid testing so I can compare results with what I would expect, nothing more

Do not expect a method tested here to have no bugs just because the test seems ok.
